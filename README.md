## 🚀 Doble Finalidad
Este proyecto está diseñado para ser un puente digital que empodera al adulto mayor en dos vertientes:

1.  **Asistente de Día a Día:** Apoyo constante mediante comandos de voz para movilidad segura (Rutas Seguras), recordatorios de salud y compañía proactiva [cite: 4, 5].
2.  **Asistente de Trámites y Programas:** Un "atajo" inteligente que simplifica el acceso a **INAPAM**, **Mi Pasaje** y otros programas del DIF Jalisco, permitiendo la carga de documentos legales para su validación oficial [cite: 1, 3, 4].

---

## 🎙️ Guía de Uso por Voz (Atajos Rápidos)
El sistema reconoce comandos naturales para evitar menús complicados:

*   **Registro:** "Quiero registrarme en el DIF" o "Quiero subir mi CURP" [cite: 6].
*   **Transporte:** "¿Cómo voy con mi apoyo de Mi Pasaje?" [cite: 6].
*   **Descuentos:** "Necesito mi tarjeta de INAPAM" o "¿Qué necesito para los descuentos?" [cite: 6].
*   **Movilidad:** "Llévame al Centro de Día Eduardo Montaño" o "Ruta segura a la UNI3" [cite: 5, 6].

---

## 🛡️ Seguridad y Gestión de Datos
La privacidad es nuestra prioridad absoluta. El sistema cumple con los siguientes estándares:

*   **Validación Documental:** Los documentos (CURP/INE) subidos por el usuario son validados directamente contra las bases del Padrón Único de Beneficiarios (PUB) [cite: 3].
*   **Acceso Restringido:** Solo personal habilitado del **DIF Jalisco** (Trabajadores Sociales y Administradores) tiene acceso a la base de datos para consultar y validar la información [cite: 1, 8].
*   **Almacenamiento Seguro:** Encriptación de extremo a extremo para proteger la identidad y documentos de los beneficiarios [cite: 1].

## 📂 Arquitectura del Proyecto
*   `index.html`: Interfaz de alta accesibilidad con botones gigantes y carga de archivos [cite: 8].
*   `admin.html`: Panel de gestión para empleados del DIF Jalisco [cite: 1].

*   `rutas_seguras.json`: Directorio geolocalizado de centros DIF, asilos y estancias en Jalisco [cite: 5].
*   `ETICA.md`: Marco legal y ético para el manejo de datos sensibles [cite: 1, 8].
