## 🚀 Doble Finalidad
1. **Asistente de Día a Día:** Apoyo en movilidad (Rutas Seguras), recordatorios y compañía por voz [cite: 4].
2. **Asistente de Trámites:** Atajo inteligente para INAPAM, Mi Pasaje y programas del DIF Jalisco, con validación de documentos (CURP) [cite: 1, 3, 4].

## 🛡️ Seguridad y Datos
- **Validación:** Los documentos (CURP/INE) se validan contra bases oficiales para el registro en el PUB [cite: 3].
- **Acceso Restringido:** Solo personal autorizado del DIF Jalisco puede consultar la base de datos de beneficiarios [cite: 1].
- **Privacidad:** Encriptación de extremo a extremo en el almacenamiento de documentos.

## 📂 Arquitectura del Repositorio
- `index.html`: Interfaz visual de alta accesibilidad con activación de voz.
- `onboarding_PUB_Maestro.json`: Lógica de conversación, requisitos de trámites y validación de perfil [cite: 3].
- `rutas_seguras_Jalisco.json`: Directorio geolocalizado de centros de atención [cite: 5].
- `ETICA.md`: Marco legal y ético para el manejo de datos sensibles [cite: 1].
