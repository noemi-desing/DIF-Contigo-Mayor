# DIF Contigo Mayor 🔘
Sistema inteligente de acompañamiento para adultos mayores en Jalisco.

## 🚀 Funcionalidades
* **Asistente de Voz:** Reconoce comandos para trámites (INAPAM, Mi Pasaje).
* **Botón SOS:** Envía alertas en tiempo real al familiar mediante WebSockets.
* **Interfaz de Alta Visibilidad:** Botones gigantes y contraste optimizado.
