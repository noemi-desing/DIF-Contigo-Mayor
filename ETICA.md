# Manual Ético y de Privacidad - DIF Contigo Mayor 🛡️

Este documento establece los principios éticos y las salvaguardas de privacidad para el asistente "DIF Contigo Mayor" en Jalisco.

## 1. Privacidad y Seguridad de Datos
*   **Validación de Identidad**: Los documentos como la CURP se utilizan exclusivamente para el registro oficial en el Padrón Único de Beneficiarios (PUB)[cite: 8].
*   **Acceso Restringido**: Solo el personal autorizado del DIF Jalisco (Trabajadores Sociales y Administradores) tiene acceso a la base de datos de los usuarios[cite: 8, 9].
*   **Encriptación**: Los documentos subidos se almacenan bajo protocolos de seguridad para evitar filtraciones.

## 2. Autonomía del Usuario
*   **Consentimiento**: La IA siempre solicitará permiso antes de procesar un trámite o acceder a la ubicación del usuario[cite: 9].
*   **Transparencia**: El sistema identificará claramente qué requisitos faltan para completar apoyos como Mi Pasaje o INAPAM[cite: 9].

## 3. Inclusión
*   **Diseño Accesible**: La interfaz de botones gigantes permite que personas con debilidad visual o dificultades motoras operen la app con facilidad.
