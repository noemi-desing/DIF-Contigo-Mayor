# Manual Ético y de Privacidad - DIF Contigo Mayor 🛡️

Este documento establece los principios éticos y las salvaguardas de privacidad que rigen el funcionamiento del asistente inteligente "DIF Contigo Mayor", diseñado para los adultos mayores del Estado de Jalisco.

## 1. Principio de Autonomía y Consentimiento
*   **Identificación Clara:** La IA siempre se identificará como un sistema automatizado al inicio de cada interacción [cite: 1].
*   **Consentimiento por Voz:** El registro en el Padrón Único de Beneficiarios (PUB) solo se iniciará tras un "Sí" explícito del usuario después de escuchar los términos de privacidad [cite: 1, 3].
*   **Derecho al Olvido:** El usuario puede solicitar, mediante comando de voz, la eliminación inmediata de sus datos personales de la sesión actual [cite: 1].

## 2. Privacidad y Seguridad de Datos
*   **Encriptación de Datos:** Toda la información recolectada (CURP, domicilio, nombre) se enviará de forma encriptada directamente a los servidores oficiales del DIF Jalisco [cite: 1].
*   **No Comercialización:** Los datos personales jamás serán compartidos con terceros ni utilizados para fines comerciales [cite: 1].
*   **Mínima Recolección:** Solo se solicitarán los datos estrictamente necesarios para el trámite que el usuario desee realizar [cite: 1].

## 3. Inclusión y Equidad Digital
*   **Adaptación Lingüística:** El sistema está diseñado para reconocer y respetar los modismos y el lenguaje cotidiano de las diversas regiones de Jalisco [cite: 1, 3].
*   **Accesibilidad Total:** La interfaz de botones gigantes de alto contraste asegura que personas con debilidad visual puedan navegar sin asistencia externa [cite: 4].

## 4. Responsabilidad y Supervisión Humana
*   **Escalación a Humanos:** Ante cualquier situación de emergencia o confusión crítica, la IA facilitará el contacto inmediato con un operador humano del DIF o servicios de emergencia [cite: 1, 4].
*   **Revisión Periódica:** Los algoritmos de respuesta serán auditados mensualmente para asegurar que no existan sesgos o discriminación en la atención [cite: 1].
